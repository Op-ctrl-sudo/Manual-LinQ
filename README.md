# Resumen
LinQ es una tecnologia desarrollada por microsoft la cual nos permite realizar consultas y manipulacion de datos usando lenguajes como C#, la cual es una herramienta poderosa que simplifica y mejora la manipulacion y consulta de datos, lo que aumenta la productividad y facilita la escritura de codigo limpio.
Fue introducida como parte de .NET y es una parte integral de lenguajes modernos de Microsoft, el que nos permite escribir consultas y manipular los datos de manera intuitiva y legible.
# Tipos de LinQ
- LINQ to Objects
- LINQ to SQL
- LINQ to XML

- Sintaxis Integrada
- Tipado Fuerte
- Optimización Automática
